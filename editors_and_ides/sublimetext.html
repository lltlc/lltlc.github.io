附录：SublimeText 编辑器¶

1. 常用快捷键¶

选择多行之后，⇧ + ⌘ + L ，再 ← 可将所有光标移动到行首；

2. 基本设置¶

Preferences.sublime-settings :

{
        "font_face": "Courier New",
        "font_size": 21,
        "highlight_line": true,
        "ignored_packages":
        [
                "RestructuredText",
                "Vintage"
        ],
        "tab_size": 2,
        "theme": "Default.sublime-theme"
}
